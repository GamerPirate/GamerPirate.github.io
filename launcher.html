What is Gamer Pirate Launcher?

Gamer Pirate Launcher is a free-to-use software that allows you to Download, Update, and Play Gamer Pirate games' from it.

What's the advantage of using the Launcher?

You get to know everything before anyone else. It also has an exclusive look on the upcoming game's information and screenshots.
It allows you to easily download any game you want and know more info about it along with the required game specs.

The Launcher won't detect a game while it's installed, how to fix this problem?

Follow these steps to fix the problem:
1- Check your game's version to see if it's compatible or not.
2- Check the game files if there is any file missing try reinstalling the game.
3- Browse the game's path to make sure it's set to the right game path.
4.0- Click "Scan" to scan for any files change.
4.1- Or go to Settings and activate "Continous Scan" option.

If the problem still exist you can contact me and I will reply to you as soon as possible.

Some images are missing, or stuck at the loading screen, what can I do?

Follow these steps to fix the problem:
1- Check your internet connection.
2- Try closing and relaunching the launcher.

If these steps didn't help then change into Offline Mode till your internet speed returns to it's normal speed. If you have a fast internet speed and you are still having this probjem then contact me and I will help you.

Note: If you have a slow internet connection the images would take longer to load.
