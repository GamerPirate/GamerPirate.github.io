F.A.Q

What is the Gamer Pirate Launcher?

Gamer Pirate Launcher is a free-to-use software that allows you to Download, Update, and Play Gamer Pirate games' from it.

What's the advantage of using the Launcher?

It allows you to easily download any game you want and know more info about it along with the required game specs.

The Launcher won't detect a game while it's installed, how to fix this problem?

Follow these steps to fix the problem:
1- Check your game's version to see if it's compatible or not.
2- Check the game files if there is any file missing try reinstalling the game.
3- Browse the game's path to make sure it's set to the right game path.
4.0- Click "Scan" to scan for any files change.
4.1- Or go to Settings and activate "Continous Scan" option.

If the problem still exist you can contact me and I will reply to you as soon as possible.

Some images are missing, or stuck at the loading screen, what can I do?

Follow these steps to fix the problem:
1- Check your internet connection.
2- Try closing and relaunching the launcher.

Note: If you have a slow internet connection the images would take longer to load.

If the problem still exist you can contact us and we will reply to you as soon as possible.

What is the required game's version to make the Launcher work properly?

Check your game version to check if it's compatible or not.
Marbles Playground: v1.2.0 / Build 3
Watch Out PC Edition: v3.1.0 / Build 6

Everything released after that should work properly.

Note: The launcher still can work with the older versions, But you will not be able to update the game from the launcher or get notified when an update releases. It's strongly recommended to update the games to the latest version.

What's the Alpha release of the launcher?

Alpha is the first stage in releasing the launhcer. It means that the launcher is still in development and not all the features planned to be in it is available, but new features are still being added.

My question is not listed here, what can I do?

You can contact me and I will reply to you as soon as possible.

I want to see a feature that's not in the Launcher at the moment

You can send me an email and I will read it ASAP, anything is really appreciated and will be read carefully.

Is that's all what the launcher can do?

Currently yes, but I'm working on adding a lot of new features to the launcher in the near feature.

The Launcher forces me to update it, why?

A new update means that you are running an old version of the launcher and you should update it to continue using it.
These updates help the launcher work better so that you get the best experience, and also adds a lot of new features, fixes, improvements, etc. So it's worth the update.
